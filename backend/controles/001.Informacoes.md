📜 Diretrizes Obrigatórias 📜
─────────────────────────────
A camada de Controles é o ponto de conexão entre as Rotas e os Serviços Backend.
A camada de Controles orquestra o fluxo de dados da requisição para a resposta.
A camada de Controles é responsável por receber os dados da requisição.
A camada de Controles é responsável por chamar os serviços apropriados.
A camada de Controles é responsável por formatar a resposta a ser enviada ao cliente.
A camada de Controles é importada pela camada de Rotas.
A camada de Controles importa a camada de Serviços Backend.
🔄 Fluxo de Dados 🔄
─────────────────────────────
Fluxo de dados começa recebendo a requisição da camada de Rotas.
Fluxo de dados continua extraindo e validando os parâmetros da requisição (body, query, params).
Fluxo de dados chama um ou mais Serviços Backend para executar a lógica de negócio.
Fluxo de dados formata a resposta (sucesso ou erro) e a retorna para a camada de Rotas.
⚡ Responsabilidades ⚡
─────────────────────────────
Responsabilidade de manipulação da requisição extrai dados da requisição HTTP.
Responsabilidade de chamada de serviços invoca os Serviços Backend com os dados corretos.
Responsabilidade de formatação da resposta estrutura o JSON de resposta para o cliente.
Responsabilidade de tratamento de erros captura erros dos serviços e os formata em uma resposta de erro HTTP.
